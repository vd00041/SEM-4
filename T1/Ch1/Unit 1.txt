JSON parse():
JSON String --> JS Object
(used when data work to recived from server)
EX-------------------------------------------------------------------------
let obj = '{ "var1": "LJ", "var2": "University" }';
console.log(obj)
console.log(obj.var1 + " " + obj.var2)

JSON stringfy():
JS OBJECT --> JSON String
(used when data work us send to server)
EX------------------------------------------------------------------------

console.log(obj)
console.log(obj.var1 + " " + obj.var2)
